## [1.0.0] - 2025-12-16

### Added
- Initial release of the JCE Dynamic Links plugin
- Dynamic configuration for multiple extensions and database tables
- Free choice of ID field, title field, and additional display fields
- State filters with customizable values
- Configurable ordering by any field
- Multiple configuration support - create unlimited link categories
- Support for Portuguese (pt-BR) and English (en-GB) translations
- Update server for automatic updates

### Features
- **Flexible Table Selection**: Choose any database table from any extension
- **Custom Fields**: Define which columns to use for ID, title, and display
- **Additional Display Fields**: Show extra information (like language, category) in parentheses
- **State Filtering**: Filter by publication state with custom field and value
- **Smart Search**: Search across title and additional fields
- **Configurable URLs**: Customize component, view, and parameter names
- **Multiple Configurations**: Create unlimited link types in one plugin
